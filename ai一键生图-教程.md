## AI一键生图是什么？

AI一键生图就是输入文字描述，AI自动生成对应的图片。你不需要会画画、不需要找素材，只需要把想要的效果描述清楚就行。

AI生图的核心优势：
- **速度快**：几秒到几十秒就能出图
- **成本低**：不需要请设计师
- **创意无限**：任何你能想到的画面都能生成
- **多风格**：写实、插画、扁平、3D等风格都能做

## AI一键生图的操作步骤

**第一步**：打开AI生图工具

**第二步**：在输入框中写下你对图片的描述（提示词）

**第三步**：选择图片比例和风格

**第四步**：点击生成，等待AI出图

**第五步**：预览效果，下载高清图片

## 提示词怎么写？

好的提示词是AI生图的关键。记住这个公式：

**主体 + 环境 + 风格 + 细节**

示例：
```
一个白色陶瓷咖啡杯放在木桌上，自然光从侧面照入，极简风格，商业摄影级
```

## AI一键生图技巧

1. **具体不要抽象**：说"红色运动鞋在跑道上"比说"一双鞋"效果好
2. **加入风格词**：商业摄影、极简主义、赛博朋克等
3. **控制画面比例**：1:1适合商品图，16:9适合海报
4. **多试几次**：同一提示词生成多次，选最满意的
