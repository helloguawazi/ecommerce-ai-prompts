# 操作说明 - 关键词驱动内容发布系统

一句话说关键词，自动生成文章并发布上线。

---

## 快速开始

### 生成单篇文章

在对话中直接说出关键词即可，例如：

> "帮我写一篇 双十一 烘焙 海报"
> "出一篇 国风 茶叶 详情页"
> "写一个 image2 怎么用的教程"

系统会自动：
1. 识别关键词归属站点
2. 匹配风格/品类/节日
3. 生成 SEO 优化文章
4. 写入文件并提交 Git
5. GitHub Pages 自动部署上线

### 批量生成

运行 Workflow 可以每天批量生成多篇：

```
关键词: "双十一 烘焙 海报"
关键词: "春节 酒水 主图"
关键词: "image2 对比 SD"
```

---

## 注意事项

1. 配图目前 Workflow 生成文字内容，配图需要手动或后续接生图 API
2. GitHub Pages 需确保仓库 Settings > Pages 已开启 main 分支部署
3. 差异化：GitHub 内容与独立站保持角度差异，避免全文一致
4. 外链：每篇文章会自动植入站间互链
5. 第一次使用需要手动 git push 一次确认仓库 remote 配置正确
